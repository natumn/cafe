# Cafe Order App

## Set Up

### Install Rust or update.
$ curl https://sh.rustup.rs -sSf | sh

### Install nightly version.
$ rustup install nightly

### Add WebAssembly target.
$ rustup target add wasm32-unknown-unknown --toolchain nightly

### start localhost.
$ rustup target add wasm32-unknown-unknown --toolchain nightly
